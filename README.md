# Phase 1 Week 3 – Functions & Type Hints

A collection of Python exercises covering core function concepts.

## Topics Covered

- **Higher-Order Functions + Lambda** (`4. Higher_Order Functions + Lambda/higher_order_lambda.py`)
  - `blink(signal_name, timer_func)`: accepts a callable to format signal-light blink rules.
  - `filter_orders(order_list, rule)`: filters a list of order amounts using a passed-in predicate (supports both named functions and inline `lambda` expressions).
