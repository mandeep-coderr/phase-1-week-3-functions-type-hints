# Phase 1 Week 3 – Functions & Type Hints

Practice exercises exploring Python functions, parameters, and return values.

## Topics Covered

- **Parameters & Return Values** (`2. Parameters_and_Return_values/practice_file.py`)
  - Default parameters (e.g. `calculate_tax(amount, tax_rate=0.18)`)
  - Boolean return values (e.g. `is_valid_email`)
  - Positional vs. keyword arguments (e.g. `format_log(level, message, timestamp="12:00")`)
